## URL-MINIFIER
Сократитель ссылок

## DevOps
Для сборки используем
`docker-compose build`

Для поднятия контейнеров
`docker-compose up -d`

Для приглушения контейнеров
`docker-compose down`

## Описание проекта

#### Container

#### DB
